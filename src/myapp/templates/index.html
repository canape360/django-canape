初心者向けにこのアプリの作り方を説明します

Daigo のアプリケーション手順から始めましょう。

Django プロジェクト作成手順

1. cd ~/projects/django-canape/src（今回は /Users/user1/projects/django-canape/srcに存在する）

2. django-admin startproject プロジェクト名

3. cd プロジェクト名

4. python3 manage.py runserver(ここではpython3 manage.py runserver 8001で起動)

コマンドを実行すると、Django 標準の admin 画面が作成されます。

同じ名前の設定用フォルダと、管理用の manage.py が自動生成されます。

設定フォルダ内の settings.py と urls.py は最重要ファイルです。

▶ プロジェクト作成の詳しい解説

アプリケーション作成

python3 manage.py startapp myapp

settings.py の INSTALLED_APPS に 'myapp' を追加します。

templates / static フォルダを作成し、HTML・CSS を管理します。

views.py で View を定義し、urls.py で URL と紐づけます。

プロジェクト側 urls.py で include() を使い、 アプリの URL を読み込みます。
